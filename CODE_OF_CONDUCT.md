# Code of Conduct

Prose follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

## Summary

Be respectful. Harassment, discrimination, and personal attacks are not tolerated — in issues, PRs, or any other project space.

## Enforcement

Report violations to **lunaleija@outlook.com**. Reports are handled confidentially.
